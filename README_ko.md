# 🎄 크리스마스 레드-블랙 트리 (지크 에디션)

> **언어 전환: [English](./README.md) | [中文版](./README_zh.md)**

**완벽하게 균형 잡힌 레드-블랙 트리** 구조를 기반으로 한 인터랙티브 크리스마스 트리 시각화입니다.

EXO의 “첫눈”에서 영감을 받았습니다.

### 🖼️ 미리보기

![미리보기](./preview.png)

### ✨ 주요 기능

- **다이내믹 물리**: 4족 보행 애니메이션이 적용된 순록과 실시간 눈 내리는 효과.
- **CS 테마**: 트리 노드가 RBT 규칙을 엄격히 따름 (블랙 루트 -> 레드 -> 블랙 -> 레드).
- **인터랙티브**: 트리 하단의 선물 노드를 클릭하면 CS 테마의 무작위 소원을 받을 수 있습니다.

### 🛠️ 사용자 정의

#### 1. 소원 문구 수정

`christmas_rbt.py` 파일에서 `WISH_POOL` 리스트를 찾아 수정하세요:

```python
WISH_POOL = ["나만의 소원 1", "나만의 소원 2"]
```

#### 2. 음악 교체

루트 폴더의 music.mp3를 원하는 MP3 파일로 교체하세요. 파일 이름은 반드시 music.mp3여야 합니다.

### 🚀 실행 파일 생성 (.exe)

필수 라이브러리 설치:

```bash
pip install pygame pyinstaller
```

빌드 명령 실행:

```bash
pyinstaller --onefile --noconsole christmas_rbt.py
```

dist/ 폴더에서 생성된 프로그램을 확인하세요. 참고: .exe 실행 시 music.mp3 파일이 같은 폴더에 있어야 합니다.
